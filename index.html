Mermaid.js Full Gamut Showcase

This page demonstrates various types of diagrams that can be created using Mermaid.js. Each section includes a rendered diagram and its corresponding Mermaid script code.

3. Class Diagram

%% Class Diagram Example classDiagram class Animal { +String name +int age +void eat() +void sleep() } class Dog { +void bark() } class Cat { +void meow() } Animal <|-- Dog Animal <|-- Cat 
classDiagram
  class Animal {
    +String name
    +int age
    +void eat()
    +void sleep()
  }
  class Dog {
    +void bark()
  }
  class Cat {
    +void meow()
  }
  Animal <|-- Dog
  Animal <|-- Cat
